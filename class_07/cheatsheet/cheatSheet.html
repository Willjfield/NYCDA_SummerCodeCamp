• Variables

    Numbers

    Numbers can hold any real number. Example declaration: var x=10; Example use: console.log(x*9);

    Strings

    Strings hold alphabetic characters. Example declaration: var s="Any text"; Example use: console.log(s);

    Arrays

    A collection of variables. Example declaration: var myArray=["some text", 1, 5.5]; Example use: alert(myArray[0]);

    Booleans

    Booleans are either true or false. Example declaration: var myBoolean=true; Example use: alert(myBoolean);

    Objects

    A complex data structure. Example declaration: var student={type:"Grad Student",age:30}; Example use: alert(student.age);

  • Debugging

    Alert

    Creates a pop-up window. Example use: alert("A message");

    Console Log

    Shows output in console. Example use: console.log(myVariable);

  • Conditionals

    Control the flow of a program. Create switches depending on values of variables.

    if

    Tests whether a condition is true or false. Example use: if(hour<12){opening="good morning"};

    if/else

    Use else when you need a process to run if the condition is not true. Example use: if(hour<12){opening="good morning"} else {opening="good evening"};

    if/else if/else

    Use else if when you want to run another test before getting to the else condition.
    Example use:
    if(hour<12){
    opening="good morning"
    }else if(hour==18){
    opening="good evening"
    }else {
    opening="good night"
    };

  • Functions

    A function an ecapsulation of code.

    No Argument

    Example declaration:
    function myFavoriteFunction(){
    alert("some alert");
    }
    Example use:
    myFavoriteFunction();

    With Argument

    Use when you want input to come into the function.
    Example declaration:
    function myFavoriteFunction(input){
    alert("hello "+input);
    }
    Example use:
    myFavoriteFunction("class");
